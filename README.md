## Задание 

Реализуйте три функции в файле `solution.js`:

### 1. `filterEven(arr)`
Принимает массив чисел, возвращает только чётные числа.
filterEven([1, 2, 3, 4, 5, 6]) // → [2, 4, 6]

### 2. `doubleAll(arr)`
Принимает массив чисел, возвращает массив где каждое число умножено на 2.
doubleAll([1, 2, 3]) // → [2, 4, 6]

### 3. `sum(arr)`
Принимает массив чисел, возвращает их сумму.
sum([1, 2, 3, 4]) // → 10
