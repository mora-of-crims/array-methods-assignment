# Array Methods Assignment

## Задание

Реализуйте три функции в файле `solution.js`:

### 1. `filterEven(arr)`
Принимает массив чисел, возвращает **только чётные** числа.

```js
filterEven([1, 2, 3, 4, 5, 6]) // → [2, 4, 6]
filterEven([1, 3, 5])           // → []
```

### 2. `doubleAll(arr)`
Принимает массив чисел, возвращает новый массив, где **каждое число умножено на 2**.

```js
doubleAll([1, 2, 3]) // → [2, 4, 6]
doubleAll([0, 5])    // → [0, 10]
```

### 3. `sum(arr)`
Принимает массив чисел, возвращает **их сумму**.

```js
sum([1, 2, 3, 4]) // → 10
sum([])           // → 0
```

## Как выполнить задание

1. Откройте файл `solution.js`
2. Напишите реализацию трёх функций
3. Запустите тесты локально: `npm test`
4. Запушьте изменения — тесты запустятся автоматически
